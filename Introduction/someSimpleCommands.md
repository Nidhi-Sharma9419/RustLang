cargo run


To write code in .rs file and execute it in Windows

1. make directory : mkdir myRustProg
2. run `cd myRustProg`
3. run `rustc myRustProg`
4. run `.\myRustProg.exe`


To write code in .rs file and execute it in macOS

1. make directory : mkdir myRustProg
2. run `cd myRustProg`
3. run `rustc myRustProg`
4. run `./myRustProg`
